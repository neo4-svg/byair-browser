# Contributing to Byair Browser

Thank you for your interest in contributing to **Byair Browser**!  
This project is written in **C** using **GTK3** and **WebKit2GTK**, and is currently in **active development stage**.  
We welcome contributions of all kinds — from bug fixes and documentation improvements to new features.

---

## 🛠 Contribution Guidelines

- **Issues**  
  - Use GitHub Issues to report bugs, request features, or suggest improvements.  
  - Please provide clear steps to reproduce bugs and as much detail as possible.

- **Pull Requests**  
  - Fork the repository and create a new branch for your changes.  
  - Keep commits focused and messages descriptive.  
  - Open a pull request with a clear explanation of what you changed and why.  

- **Code Style**  
  - Stick to clean, readable C code.  
  - Follow existing formatting conventions in the project.  
  - Add comments where necessary to explain complex logic.

- **Documentation**  
  - Update README or inline comments if your changes affect usage or design.  
  - Keep explanations beginner‑friendly where possible.

---

## ⚠️ Development Stage Notice
Byair Browser is still evolving. Expect bugs, missing features, and rapid changes.  
The **update mechanism** will be published once the best approach is determined.  
Contributors are encouraged to help shape this process.

---

## 🚨 Security Disclaimer
Byair Browser is an open‑source project.  
If you submit or include **malicious code, exploits, or harmful contributions**, the maintainers are **not responsible** for any resulting damage.  
All contributions are reviewed, but responsibility for safe usage lies with the user.
