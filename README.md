# Byair Browser

Byair Browser is a lightweight, privacy‑focused web browser written in **C** using **GTK3** and **WebKit2GTK**.  
It is designed to provide a clean interface, essential browsing features, and sensible privacy defaults.

## ✨ Features
- [Tabbed browsing](ca://s?q=add_tabbed_browsing_in_Byair_Browser) with `GtkNotebook`  
- [DuckDuckGo search integration](ca://s?q=integrate_DuckDuckGo_search_in_Byair_Browser) by default  
- [Navigation controls](ca://s?q=add_navigation_controls_in_Byair_Browser): Back, Forward, Reload, Home  
- [Branded homepage](ca://s?q=create_branded_homepage_in_Byair_Browser) with shortcuts to GitHub, Hacker News, and Reddit  
- [Privacy defaults](ca://s?q=privacy_settings_in_Byair_Browser): blocks third‑party cookies, disables file access from file URLs  

## 🎯 Design Philosophy
Byair Browser is built around three principles:
1. **Minimalism** — a clean, distraction‑free interface.  
2. **Privacy** — strong defaults that protect users.  
3. **Extensibility** — a codebase that can be expanded with new features.
